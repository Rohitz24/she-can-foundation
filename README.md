# She Can Foundation - Frontend Development Task

## Overview

This project is a responsive single page website created for the She Can Foundation Frontend Development Internship Task.

The website is designed to promote women's empowerment through education, skill development, leadership, and community support initiatives.

## Features

* Responsive Design
* Modern Landing Page UI
* Animated Gradient Background
* Dark Mode Toggle
* Glassmorphism Cards
* Statistics Counter Animation
* Testimonials Section
* Contact Form
* Smooth Scrolling
* AOS Scroll Animations
* Mobile Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* AOS (Animate On Scroll) Library
* Google Fonts (Poppins)

## Project Structure

SheCanFoundation/

├── index.html

├── style.css

├── script.js

└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in any modern web browser.
